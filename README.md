# Admin Dashboard

Admin dashboard with authentication and user CRUD operations.

## Features
- Login/Logout (JWT)
- User list, Add, Edit, Delete
- Route protection
- Responsive UI

## Tech Stack
Vue 3 · Vite · Pinia · Vue Router · Axios · Tailwind CSS

## API
DummyJSON (https://dummyjson.com)

## Run
npm install
npm run dev
